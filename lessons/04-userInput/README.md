# Lesson 04 - User Input & DOM Interaction

## 🎯 Objective

Learn how to capture user input from an HTML form and use JavaScript to update the webpage dynamically.

---

## 🧠 What this project demonstrates

This project shows how a website can:

- Accept user input
- React to button clicks
- Update content dynamically
- Provide personalized responses

---

## 🌍 Real-world applications

This concept is used in:

- Login systems (Welcome back, user)
- Search engines
- Chat applications
- Forms and surveys
- AI input systems

---

## 🔁 How it works (flow)

1. User types name into input box
2. User clicks submit button
3. JavaScript reads input value
4. Webpage updates greeting text

---

## 🧠 Key takeaway

JavaScript is not just for calculations — it is used to make websites interactive and responsive to users.
