# Lesson 06 - Constants (const)

## 🎯 Objective

Understand how to use constants in JavaScript to store values that should never change.

---

## 🧠 What is a constant?

A constant is a variable whose value cannot be reassigned after it is created.

---

## 🌍 Real-world usage

Constants are used for:

- Mathematical values (PI)
- Configuration settings
- API endpoints
- Business rules (tax rates)

---

## 🧪 Project example

We built a circle calculator that:

- Accepts user input (radius)
- Uses a constant (PI)
- Calculates circumference
- Displays result dynamically

---

## 📌 Formula used

C = 2 × π × r

---

## 🧠 Key takeaway

Use const when the value should never change, and use let when the value must change during program execution.
