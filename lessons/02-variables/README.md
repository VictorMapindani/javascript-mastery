# Lesson 02 - JavaScript Variables

##  Objective
Understand how to store and manage data using variables in JavaScript.

---

## Concepts Covered
- `let`
- `const`
- Variables in memory
- Reassigning values
- Basic objects

---

## Real-World Understanding

Variables are used in every application to store:

- User data
- API responses
- Application state
- Configuration values

---

## Example Use Case

A developer profile object is created using variables:

- Name
- Age
- Role
- Tech stack

---

## Key Takeaway

- `const` = value should not change
- `let` = value can change
- Variables are the foundation of all programming
