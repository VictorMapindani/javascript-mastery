# Lesson 03 - Data Types & Basic Operators

## Objective
Understand how JavaScript stores data and how we manipulate it using operators.

---

## Concepts Covered

### Data Types
- String
- Number
- Boolean
- Null
- Undefined

### Reference Types
- Objects
- Arrays

### Operators
- Arithmetic operators (+ - * / % **)
- Shorthand operators
- Increment / Decrement
- Operator precedence

---

## Real-World Usage

These concepts are used in:

- User profiles (objects)
- Product lists (arrays)
- Pricing systems (arithmetic)
- Analytics dashboards
- API data handling

---

## Key Takeaway

Data types define what data is, and operators define how we manipulate that data in real applications.
